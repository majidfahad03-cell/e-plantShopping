# e-plantShopping

e-plantShopping is a React plant shopping application for Paradise Nursery.

The project includes a landing page, company information, a product listing page, and a shopping cart page. Users can view plants, add items to the cart, remove items, and update quantities.

Main features:
- Paradise Nursery landing page
- About Us page
- Product list
- Shopping cart
- Add, remove, and update cart items
- Redux state management
